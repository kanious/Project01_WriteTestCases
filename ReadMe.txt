- How to build and run the project.
  There are 2 projects in the solution.
  (1) BlackBox Project
      A DLL project. Automatically connected with the main project.
  (2) TestCases Project
      The main project. Set as a startup project.

  Just build solution and run with Debug or Release configuration mode.
  Or run .exe file after building solution
  .exe file location : (SolutionFolder)/(Platform)/(ConfigurationFolder)


- User Input Options
  There are two input commands.
  1. TEST : Write '1' and enter. The test cases will be executed.
  2. QUIT : Write '2' and enter. Program will be terminated.
